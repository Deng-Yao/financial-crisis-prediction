# 数据获取指南

## 📋 数据源介绍

### CSMAR数据库（推荐）

**CSMAR**（China Stock Market & Accounting Research Database）是中国学术界最常用的金融数据库，由深圳国泰安教育技术股份有限公司开发。

**官网**：https://data.csmar.com/

**优势**：
- 数据质量高，经过严格校验
- 学术界广泛认可
- 字段说明详细
- 支持批量下载

## 📥 CSMAR数据下载步骤

### 第一步：注册账号

1. 访问 https://data.csmar.com/
2. 点击"注册"
3. 填写个人信息（建议使用学校邮箱）
4. 完成邮箱验证

### 第二步：登录系统

1. 使用注册的账号密码登录
2. 如果是学校用户，可以通过IP认证自动登录

### 第三步：下载数据表

#### 3.1 上市公司基本信息表

```
路径：公司研究 → 上市公司基本信息
文件名：STK_LISTEDCOINFOANL.csv

下载设置：
- 证券类型：A股
- 字段：全选
- 格式：CSV
- 编码：UTF-8
```

**关键字段**：
- Symbol：股票代码
- ShortName：股票简称
- EndDate：统计截止日期
- IndustryCodeD：行业代码（中国上市公司协会分类）
- IndustryNameD：行业名称

#### 3.2 资产负债表

```
路径：财务报表 → 资产负债表
文件名：FS_Combas.csv

下载设置：
- 证券类型：A股
- 报表类型：合并报表（Typrep = 'A'）
- 时间范围：2003-2024年
- 字段：全选
- 格式：CSV
- 编码：UTF-8
```

**关键字段**：
- Stkcd：证券代码
- Accper：统计截止日期
- Typrep：报表类型
- A001000000：资产总计
- A002000000：负债合计
- A003000000：所有者权益合计

#### 3.3 利润表

```
路径：财务报表 → 利润表
文件名：FS_Comins.csv

下载设置：
- 证券类型：A股
```

#### 3.4 现金流量表（直接法）

```
文件名：FS_Comscfd.csv

下载设置：
```

#### 3.5 特殊处理变动文件

```
路径：公司研究 → 特殊处理与特别转让
文件名：SPT_Trdchg.csv

下载设置：
- 证券类型：A股
- 时间范围：2003-2024年
- 字段：全选
- 格式：CSV
- 编码：UTF-8
```

**关键字段**：
- Stkcd：证券代码
- Chgtype：变动类型
- Annoudt：变动公布日期

**变动类型说明**：
| 代码 | 含义 |
|------|------|
| AB | 从正常变为ST |
| BA | 从ST恢复正常 |
| AD | 从正常变为*ST |
| DA | 从*ST恢复正常 |
| BD | 从ST变为*ST |
| DB | 从*ST变为ST |

## 🔍 数据质量检查

### 1. 检查文件完整性

```python
import pandas as pd

# 检查每个文件
files = ['STK_LISTEDCOINFOANL.csv', 'FS_Combas.csv', 'FS_Comins.csv', 
         'FS_Comscfd.csv', 'SPT_Trdchg.csv']

for f in files:
    df = pd.read_csv(f'data/{f}', encoding='utf-8')
    print(f"{f}: {len(df)} 行, {len(df.columns)} 列")
```

### 2. 检查时间范围

```python
# 检查数据时间范围
df = pd.read_csv('data/FS_Combas.csv', encoding='utf-8')
df['Accper'] = pd.to_datetime(df['Accper'])
print(f"时间范围: {df['Accper'].min()} ~ {df['Accper'].max()}")
```

### 3. 检查缺失值

```python
# 检查关键字段缺失值
key_fields = ['Stkcd', 'Accper', 'Typrep', 'A001000000', 'A002000000']
for field in key_fields:
    missing = df[field].isnull().sum()
    print(f"{field}: {missing} 个缺失值 ({missing/len(df)*100:.2f}%)")
```

### 4. 检查报表类型

```python
# 确保只包含合并报表
typrep_counts = df['Typrep'].value_counts()
```

## ⚠️ 常见问题

### Q1: 下载速度很慢怎么办？

A: 
- 避开高峰期（晚上8-11点）
- 使用学校网络（通常有专线）
- 分批下载（按年份）

### Q2: 字段名称与代码不匹配？

A: 
- CSMAR可能会更新字段名称
- 检查Excel字段说明文件
- 根据实际字段名修改代码

### Q3: 数据量太大无法下载？

A: 
- 按年份分批下载
- 只下载必需的字段
- 使用CSMAR的在线查询功能

### Q4: 没有CSMAR访问权限？

A: 
- 通过学校图书馆申请
- 联系CSMAR销售申请试用
- 使用替代数据源（Wind、巨潮资讯）
